# 💰 Controle de Gastos

Uma aplicação web premium para controle financeiro pessoal, desenvolvida para a AV1 de Programação Web.

## 🚀 Funcionalidades

- **Dashboard Financeiro**: Visualização rápida do saldo atual, total de receitas e despesas.
- **Lançamento de Transações**: Formulário para registrar novas receitas ou despesas com validação inteligente.
- **Histórico Completo**: Tabela responsiva exibindo todas as movimentações.
- **Exclusão de Registros**: Capacidade de remover transações indesejadas com atualização em tempo real do saldo.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura com marcação semântica (`<header>`, `<main>`, `<section>`, `<footer>`).
- **CSS3 (Custom & Flexbox)**: Estilização moderna, variáveis nativas e layouts flexíveis.
- **Bootstrap 5**: Utilizado para o sistema de Grid, Cards e elementos visuais base.
- **JavaScript (Vanilla)**: Manipulação do DOM, gestão de estado, validação de inputs e LocalStorage.

## ⚙️ Como Executar

Não há necessidade de servidor ou dependências locais complexas:

1. Clone o repositório ou faça o download dos arquivos.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. Comece a lançar suas transações!
